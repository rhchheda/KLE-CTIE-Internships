🌐 Portal URL (Critical)
This must be your Web App /exec URL.
It looks like: https://script.google.com/macros/s/AKfycb.../exec
Get it from: Apps Script → Deploy → Manage deployments → copy URL.
All navigation links use this URL. If wrong, clicks go to "unable to open file".
🏭 Logo Override (optional)
Both logos already ship embedded in the source (/assets, no Google Drive involved) and are used by default -- you don't need to set anything here. Only fill these in if you want to swap in a different image without editing code: paste any publicly accessible image URL (a GitHub raw link, etc.) or a Google Drive share link. Leave blank to keep the built-in logos.

Pasting a Google Drive link? A normal "Anyone with the link" share URL (the one with /file/d/…/view) is a web page, not an image file, so it won't display as-is -- this is automatically rewritten into a working direct-image link when you save. The preview below updates once you click away from the field.
📹 Interview & Google Meet
Google Meet links are not auto-generated by this portal.
When a startup schedules an interview, they paste the Meet URL manually in the Schedule Interview form. The system then emails the link to the student.

To create a Meet link: go to meet.google.com → click "New meeting" → copy the link → paste it when scheduling.
⚙ First-time Setup Checklist
1
Create a new Google Sheet. Copy the Sheet ID from the URL (the long string between /d/ and /edit). In the Apps Script editor, go to Project Settings → Script Properties → add SHEET_ID. (Never hardcode it in source -- this project is meant to be version-controlled.)
2
In Apps Script editor: run setupSheets() then setupTriggers() then seedAll() (for test data).
3
Deploy as Web App: Execute as Me, Access Anyone. Copy the /exec URL.
4
Paste the /exec URL in Portal URL above and save.
5
Optional: override the built-in logos above (Drive share link or any public image URL -- Drive links are converted automatically).
6
Run checkAllFiles() from the Apps Script editor. All 12 HTML files should show OK.
7
Test: open the /exec URL in a new tab. Home page should load with seed data stats.
8
Open the Admin page and log in -- the first PIN you enter becomes the admin PIN. Do this before sharing the portal URL with anyone.
